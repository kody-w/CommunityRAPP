Demo Tips

  • Open M365 Copilot and select the Contract Analysis Agent
  • Click the Copy button to copy each message
  • Paste into the Copilot chat and press Enter
  • Wait for the response before moving to the next step
  • Step 2 generates a downloadable PDF report - click the link to show it
  • For best demo impact, pause after each response to highlight key findings
