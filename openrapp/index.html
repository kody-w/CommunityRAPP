Input: Discovery Transcript 
Customer: We handle 500 contracts monthly.
We need to track renewals and flag risky clauses.

Agent: What systems do you use?

Customer: Salesforce for CRM and DocuSign for signatures.

Agent: What's your biggest pain point?

Customer: Missing renewal deadlines.
Manual clause review takes forever.
Output: Production Agent 
class ContractTrackerAgent(BasicAgent):
    def __init__(self):
        self.name = 'ContractTracker'
        self.metadata = {
            "description": "Track contracts...",
            "parameters": {
                "action": {
                    "enum": [
                        "search_contracts",
                        "get_renewals",
                        "flag_risky_clauses"
                    ]
                }
            }
        }
